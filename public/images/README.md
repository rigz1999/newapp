# Images Directory

## Dashboard Screenshot

Place your dashboard screenshot here with the name: `dashboard-screenshot.png`

This image will be displayed on the landing page in the "Zéro Excel, Zéro erreur" section.

**File requirements:**
- Name: `dashboard-screenshot.png`
- Format: PNG or JPG
- Recommended size: 1920x1080 or similar aspect ratio
- Location: `/public/images/dashboard-screenshot.png`
